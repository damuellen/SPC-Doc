Protocol Parameterizable 

protocol Parameterizable

The Parameterizable protocol serves as a blueprint for types that need to handle and parameterize associated Codable types. It includes an associated type Parameter that must conform to the Codable protocol. Additionally, the protocol defines a static property parameter of type Parameter that can be accessed and modified, and a static function parameterize(_:) to set the value of the associated parameter.
