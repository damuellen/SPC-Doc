end​OfPrefix(while:​) 

@inlinable
  internal func endOfPrefix(
    while predicate: (Element) throws -> Bool
  ) rethrows -> Index

Returns the exclusive upper bound of the prefix of elements that satisfy the predicate.

O(n), where n is the length of the collection.

Parameters

Name Type Description
predicate (Element) throws -> Bool

A closure that takes an element of the collection as its argument and returns true if the element is part of the prefix or false if it is not. Once the predicate returns false it will not be called again.

indexed() 

@inlinable
  public func indexed() -> Indexed<Self>

Returns a collection of pairs (i, x), where i represents an index of the collection, and x represents an element.

This example iterates over the indices and elements of a set, building an array consisting of indices of names with five or fewer letters.

let names: Set = ["Sofia", "Camilla", "Martina", "Mateo", "Nicolás"]
var shorterIndices: [Set<String>.Index] = []
for (i, name) in names.indexed() {
    if name.count <= 5 {
        shorterIndices.append(i)
    }
}

Returns: A collection of paired indices and elements of this collection.
