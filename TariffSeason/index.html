struct TariffSeason: Codable

A data structure representing the tariff seasons for an energy provider.

The TariffSeason struct defines the seasons for which specific tariffs are applicable. It includes information about peak and off-peak periods on weekdays, Saturdays, and holidays. The struct is used to determine the pricing and rates for different time periods and days throughout the year.

The struct contains the following properties:

  • name: The name of the tariff season.

  • v1PfD, v1PlD, v2PfD, v2PlD: Integer values representing time periods for peak and off-peak rates for various tariffs.

  • weekday: An array of integers representing weekdays (Sunday to Saturday) when the tariff season is applicable.

  • saturday: An array of integers representing Saturdays when the tariff season is applicable.

  • holyday: An array of integers representing holidays when the tariff season is applicable.
